# Cypress Ecommerce Test Automation

Proyecto de automatización de pruebas E2E utilizando Cypress y Page Object Model.

## Tecnologías
- Cypress
- JavaScript
- Page Object Model
- Git
- GitHub

## Escenarios automatizados

1. Login exitoso
2. Login con error
3. Compra de producto
4. Agregar producto al carrito
5. Checkout completo

## Estructura del proyecto

cypress
 ├── e2e
 │   ├── compra.cy.js
 │   ├── loginError.cy.js
 │
 ├── pageObjects
 │   ├── loginPage.js
 │   ├── inventoryPage.js

## Instalación

npm install

## Ejecutar Cypress

npx cypress open
